# 龍城風格科幻小說生成計劃

## 項目概述

依方想《龍城》風格創作長篇科幻小說，融合科幻+古武+機甲元素，至少500章，每章2000-3000字，每章獨立文件。注重冷靜克制的敘事，主角從殘酷背景成長，避免老套劇情。

## 創作原則（《龍城》風格）

### 核心元素
- **類型混合**：科幻 + 古武 + 機甲
- **主角設定**：殺手/殘酷訓練營出身，冷淡理性，內心掙扎
- **世界觀**：完整等級體系，嚴謹技能訓練，制度與倫理限制
- **敘事風格**：冷靜克制，技術細節具體，內心戲通過行為展現
- **成長模式**：階梯性成長，有代價和挫折，倫理困境

### 避免的套路
- ❌ 天選之子、血脈覺醒
- ❌ 外掛式成長、無限制強大
- ❌ 打怪升級、無腦爽文
- ❌ 設定混亂、邏輯自洽差
- ❌ 工具人配角、無深度人物

## 文件結構

```
document/
├── dragon_city_structure.md          # 小說總體架構、世界觀、人物設定
├── chapter_001.txt                   # 第一章
├── chapter_002.txt                   # 第二章
├── ...                               # 後續章節
└── dragon_city_index.md              # 章節索引（可選）
```

## 實施步驟

### 階段一：建立基礎架構

1. 創建 `dragon_city_structure.md` 文件，包含：
   - 世界觀設定（腦控時代、光甲體系、師士等級）
   - 主角初始設定（殺手背景、冷淡性格、內心掙扎）
   - 故事主線大綱（訓練→考核→戰鬥→覺醒）
   - 重要配角設定（師徒、競爭者、盟友）
   - 技能訓練體系（明微法、神經反射、光甲操作）

### 階段二：生成初始章節

2. 首批生成10-20章作為開篇：
   - 建立主角殘酷的訓練營背景
   - 展現冷靜理性的性格特點
   - 引入光甲和技能訓練
   - 設定第一個主要衝突或考核

### 階段三：批量生成機制

3. 建立章節生成模板和規範：
   - 每章開頭標題格式：`第X章 章節名稱`
   - 保持冷靜克制的敘事風格
   - 每章包含完整的訓練/戰鬥/內心戲
   - 結尾適當留懸念和成長線索

4. 後續可依需求批量生成更多章節

## 技術規範

- **文件命名**：`chapter_XXX.txt`（XXX為三位數章節號，如001, 002...）
- **編碼**：UTF-8
- **每章字數**：2000-3000字
- **風格**：冷靜克制，技術細節豐富，內心掙扎真實

## 創作重點

### 世界觀設計
- **光甲體系**：機甲裝甲結合武術訓練
- **等級制度**：師士等級、技能劃分
- **訓練方法**：神經反射、呼吸錘、明微法等
- **倫理限制**：AI界限、情感測試、人性考驗

### 角色塑造
- **主角**：殺手出身，冷淡但內心有掙扎
- **配角**：有獨立性和深度，不只是背景板
- **關係**：師徒、競爭、盟友的複雜動態

### 敘事技巧
- **節奏**：日常訓練 + 衝突爆發 + 內心思考
- **戰鬥**：技術細節 + 戰術分析 + 代價呈現
- **成長**：階梯性進步，有挫折和代價

## 注意事項

- 保持《龍城》風格的冷靜克制感
- 注重技術和技能的科學性解釋
- 避免老套劇情，強調內心掙扎和倫理困境
- 確保每章內容完整，可獨立閱讀但保持連續性
- 配角要有獨立性，主角成長要有真實代價
