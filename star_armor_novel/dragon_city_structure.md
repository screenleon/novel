# 星甲：龍城風格科幻小說架構

## 世界觀總覽

### 核心設定：腦控時代的星際武道

**時代背景**：
- 公元2157年，人類進入「腦控時代」
- 腦機接口技術將人類意識與機甲完美融合
- 星際殖民時代，地球聯盟統治數十個行星系
- 武道傳統與科技進步的完美融合

**核心概念**：
- **星甲 (Star Armor)**：人類的新軀體，結合機甲裝甲與武術修煉
- **腦控系統**：意識直接操控機甲，無需物理介面
- **武道修煉**：傳統武術與神經反射訓練的結合
- **等級制度**：師士等級劃分，決定社會地位與權利

### 社會結構

**等級體系**：
- **見習師士 (Apprentice)**：基礎訓練者，穿戴輕型星甲
  - **見習初級**：剛入學，基礎操作不熟練，反應時間>0.8秒
  - **見習中級**：掌握基礎操作，通過第一次考核，反應時間0.5-0.8秒
  - **見習高級**：熟練操作，通過第二次考核，反應時間0.3-0.5秒
  - **見習巔峰**：準備晉升初階師士，反應時間<0.3秒
- **初階師士 (Initiate)**：通過考核，可參與實戰
- **中階師士 (Adept)**：精英戰士，可指揮小隊
- **高階師士 (Master)**：戰略指揮官，參與星際戰爭
- **宗師 (Grand Master)**：傳說級存在，守護人類文明

**職業劃分**：
- **訓練官 (Instructor)**：教授武道與機甲操作
- **星甲技師 (Armor Technician)**：維護與改進星甲
- **神經鍛鍊師 (Neural Trainer)**：開發神經反射能力
- **戰術分析官 (Tactical Analyst)**：研究戰鬥數據與策略
- **星際仲裁者 (Stellar Arbiter)**：處理星甲相關糾紛
- **星甲藥劑師 (Star Armor Pharmacist)**：專注於藥劑製作與研發
- **神經醫療師 (Neural Medic)**：專注於神經系統治療與修復
- **戰鬥醫療官 (Combat Medic)**：戰場急救與創傷處理

**價值觀**：
- **武道至上**：實力決定地位，訓練決定命運
- **榮譽守則**：戰鬥中的誠實與尊重
- **犧牲精神**：為聯盟利益可付出一切
- **技術創新**：不斷改進星甲與訓練方法

**見習初級**（第1-4週）：
- **基礎能力**：剛入學，基礎操作不熟練
- **反應時間**：>0.8秒

- **星甲操作**：基礎操作不穩定，情緒容易波動
- **成長速度**：快速進步，每週都有明顯變化

**見習中級**（第5-8週）：
- **基礎能力**：掌握基礎操作，通過第一次考核
- **反應時間**：0.5-0.8秒
- **明微十二法**：第一法熟練，第二法（情緒控制）初學
- **星甲操作**：基礎操作穩定，開始理解冷靜的重要性
- **成長速度**：快速進步，但開始遇到瓶頸

**見習高級**（第9-12週）：
- **基礎能力**：熟練操作，通過第二次考核

- **明微十二法**：第二法熟練，第三法（神經同步）初學
- **星甲操作**：觀察能力突出，實戰技巧提升
- **成長速度**：進步速度開始放緩，需要更多時間練習

**見習巔峰**（第13-16週）：
- **基礎能力**：準備專科選擇
- **反應時間**：<0.3秒
- **明微十二法**：第三法熟練，專科方向技能初學
- **星甲操作**：專科方向能力開始展現
- **成長速度**：進步明顯放緩，需要反覆練習才能看到微小進步

### 成長速度遞減設定

**核心原則**：前期成長快速，後期成長困難，形成巨大反差

**見習階段（第一學年）**：
- **成長速度**：快速
- **時間跨度**：4個月（16週）完成見習階段
- **表現**：每週都有明顯進步，容易看到成果

**初階階段（第二學年）**：
- **成長速度**：中等
- **時間跨度**：需要1-2年才能從初階晉升中階
- **表現**：需要數月才能看到明顯進步，一個學期可能只提升一個小等級

**中階階段（第三學年及以後）**：
- **成長速度**：緩慢
- **時間跨度**：需要3-5年甚至更長時間
- **表現**：需要數年才能看到明顯進步

**高階階段**：
- **成長速度**：極慢
- **時間跨度**：可能需要數十年，甚至終生無法達到宗師
- **表現**：微小的進步都需要大量時間和努力

### 官方測試評估系統

**核心原則**：官方測試不依賴特定功法，而是通過標準化的綜合能力評估。所有職業都需通過共同基礎測試，但各職業有專業測試項目和不同的評分權重。

#### 共同基礎測試（所有職業必考）

1. **基礎能力測試**（標準化設備測試）：
   - **反應速度測試**：標準化反應測試設備，測試神經反射速度
   - **準確度測試**：動作精準度、命中率測試
   - **穩定性測試**：長時間操作的穩定性，能量流控制穩定性
   - **神經同步度測試**：腦控系統的同步率，意識與星甲的融合度

2. **理論知識測試**：
   - **星甲技術知識**：動力系統、材料學、維修基礎
   - **安全規範**：操作規範、安全意識

#### 各職業專業測試項目與評分權重

**戰鬥職業（訓練官、實戰師士）**：
- **專業測試**：
  - 實戰能力測試（模擬戰鬥、團隊配合、應變能力）
  - 戰術指揮測試（小隊指揮、戰術制定）
- **評分權重**：基礎能力30% + 實戰能力50% + 理論知識20%

**技術職業（星甲技師）**：
- **專業測試**：
  - 星甲維修測試（故障診斷、零件更換、系統調試）
  - 技術創新測試（改裝設計、性能優化）
- **評分權重**：基礎能力20% + 專業技術60% + 理論知識20%

**神經鍛鍊師**：
- **專業測試**：
  - 神經開發測試（意識控制、感知擴張、情緒調節）
  - 訓練指導測試（設計訓練方案、指導他人）
- **評分權重**：基礎能力30% + 神經開發50% + 理論知識20%

**戰術分析官**：
- **專業測試**：
  - 數據分析測試（戰鬥數據分析、戰術評估）
  - 策略制定測試（戰術規劃、風險評估）
- **評分權重**：基礎能力20% + 分析能力60% + 理論知識20%

**星際仲裁者**：
- **專業測試**：
  - 法律知識測試（星甲相關法規、糾紛處理）
  - 判斷力測試（案例分析、公正裁決）
- **評分權重**：基礎能力20% + 法律能力50% + 判斷力30%

**藥劑師/醫療職業**：
- **專業測試**：
  - 藥劑製作測試（藥劑配方、製作工藝、品質檢測）
  - 醫療診斷測試（創傷診斷、神經損傷評估、治療方案）
- **評分權重**：基礎能力20% + 藥劑製作40% + 醫療診斷40%

#### 綜合評估

- **多項指標綜合評分**：各項測試按權重計算總分
- **等級標準線**：達到標準線即可晉升，不依賴特定功法掌握程度
- **客觀評估**：測試結果由標準化設備和評委共同評定

**測試特點**：
- **標準化**：所有學生使用相同的基礎測試標準和設備
- **專業化**：各職業有針對性的專業測試項目
- **客觀性**：不依賴特定功法，評估綜合能力
- **公平性**：不同職業和訓練方法的學生都能公平評估

### 實力評估方式

1. **考核結果明確化**：每次考核後明確說明等級變化
2. **訓練表現具體化**：通過實際表現展現進步，而非機械化數據
3. **與他人對比**：明確說明與同學的實力差距和排名
4. **階段性評估**：每週或每月加入自我評估或導師評估
5. **自然體現成長速度**：通過實際的進步幅度和時間跨度來體現，不需要通過對話說明
6. **官方測試為主**：等級晉升以官方測試結果為準，不依賴特定功法掌握程度

### 藥劑系統

**核心概念**：在腦控時代，藥劑技術與星甲技術並行發展，用於增強戰鬥能力、治療創傷、修復神經損傷。

#### 藥劑類型

**增強型藥劑**：
- **神經反應增強劑**：短期提升反應速度（副作用：疲勞）
- **能量恢復劑**：快速恢復星甲能量（限制：每日使用次數）
- **感知擴張劑**：暫時增強感知範圍（副作用：頭痛）
- **同步率提升劑**：短期提升腦控同步率（風險：過度使用會損傷神經）

**恢復型藥劑**：
- **創傷治療劑**：治療戰鬥創傷、骨折、內傷
- **神經修復劑**：修復神經損傷、腦控過度使用後遺症
- **體力恢復劑**：快速恢復體力，消除疲勞
- **精神穩定劑**：治療戰鬥創傷後遺症、心理創傷

**特殊藥劑**：
- **適應性藥劑**：幫助新星甲使用者快速適應（見習階段使用）
- **突破輔助劑**：輔助等級突破（風險高，需謹慎使用）

#### 藥劑製作系統

**材料來源**：
- 星際植物：各種行星上的特殊植物提取物
- 礦物：稀有礦物中的活性成分
- 生物提取物：特定星際生物的組織提取

**製作工藝**：
- 需要專業設備：精密溫度控制、時間掌握、純度檢測
- 製作難度：初級、中級、高級、完美級
- 品質等級：影響藥效和副作用程度

**使用限制**：
- **副作用**：所有增強型藥劑都有副作用，需要專業指導
- **風險管理**：過度使用會造成永久性損傷
- **倫理約束**：某些藥劑僅限特定場合使用
- **法律規範**：私自製作和使用違禁藥劑屬違法行為

#### 藥劑師職業分類

**星甲藥劑師 (Star Armor Pharmacist)**：
- 專注於藥劑製作與研發
- 研究新配方，改進現有藥劑
- 為戰鬥人員提供增強型藥劑

**神經醫療師 (Neural Medic)**：
- 專注於神經系統治療與修復
- 診斷和治療腦控相關的神經損傷
- 研發神經修復技術

**戰鬥醫療官 (Combat Medic)**：
- 戰場急救與創傷處理
- 快速診斷和治療戰鬥創傷
- 在戰場上提供即時醫療支援

### 倫理與限制

**星甲守則**：
- 不可濫用腦控能力操控他人意識
- 戰鬥必須遵循榮譽規則
- 星甲技術僅用於保護人類文明
- 神經強化有健康風險限制

**藥劑使用規範**：
- 增強型藥劑需在專業指導下使用
- 禁止使用違禁藥劑和過度依賴藥劑
- 藥劑副作用必須明確告知使用者
- 私自製作違禁藥劑屬違法行為

**社會約束**：
- 等級考核必須通過官方認可
- 私自改裝星甲屬違法行為
- 訓練營畢業生需接受心理評估
- 宗師級存在受聯盟特殊保護

## 章節命名規則

格式：`第X章：主題描述`
範例：
- 第1章：逃亡的開始
- 第5章：星甲的召喚
- 第15章：訓練營的陰影
- 第50章：考核的考驗
- 第100章：戰場的覺醒

命名原則：
- 反映該章核心事件或主角心境
- 避免劇透重要情節轉折
- 保持簡潔有力（4-8字）
- 呈現主角成長軌跡的邏輯發展
