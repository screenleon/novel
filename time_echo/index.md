# 《時間回音》章節索引

## 小說概述

一部關於時間感知能力的奇幻小說，主角林辰擁有罕見的裂縫感知能力，在時間河流的世界中探索未知，面對即將到來的災變。

**總章節目標**：500章以上
**每章字數**：2000-3000字
**當前進度**：5/500章
